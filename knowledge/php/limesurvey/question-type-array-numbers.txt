This question types allows survey administrators to a create large arrays with numbered dropdown boxes with a set of subquestions on the "y-axis", and another set of subquestions on the "x-axis".


<center>[[File:DP_MFN.jpg]]</center>

You can also set a [[Question type - Array (Numbers)#Checkbox layout (multiflexible_checkbox)|special question attribute]] to use this question type with checkboxes:


<center>[[File:DP_MFN_CB.jpg]]</center>
