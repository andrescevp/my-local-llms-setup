{{QS:mandatory}}

{{Alert|Enabling the '''mandatory''' attribute in a ranking question type forces your respondents to assess all the answer options. If you wish to make your respondents rank a minimum and/or maximum number answer options, use the [[Question type - Ranking#Minimum answers (min_answers)|minimum answers]] and [[Question type - Ranking#Maximum answers (max_answers)|maximum answers]] question attributes to set the lower and upper limits.}}
